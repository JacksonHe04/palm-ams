## 在端口启动，进行测试
``` bash
source ~/.bashrc
conda activate palm
cd /home/ubuntu/back-django
python manage.py runserver 0.0.0.0:8001
```
control + c 关闭

## 使用 uwsgi + nginx 启动
``` bash
source ~/.bashrc
conda activate palm
cd /home/ubuntu/back-django
/home/ubuntu/anaconda3/envs/palm/bin/uwsgi --ini /home/ubuntu/back-django/palm.ini
sudo nginx -s reload
```

## 关闭
``` bash
pkill -f uwsgi -9
```
